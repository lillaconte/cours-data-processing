## xan

> Guillaume Plique, Béatrice Mazoyer, Laura Miguel, César Pichon, Anna Charles, & Julien Pontoire. (2025). xan, the CSV magician. (0.50.0). Zenodo. https://doi.org/10.5281/zenodo.15310200

### What is xan?

[xan](https://github.com/medialab/xan) is a fast, composable CLI tool for working with CSV data in the terminal, built by médialab (Sciences Po) in Rust. It bundles commands for filtering, joining, aggregating, and previewing CSV files, plus its own expression language. It also handles scraping.

#### xan scrape command

`xan scrape` is a command designed to extract structured data from web pages. It allows users to:
- extract information using selectors (HTML, XML, etc.);
- convert unstructured web content into tabular (CSV-like) data;
- integrate scraping directly into command-line data workflows.

> Documentation: https://github.com/medialab/xan/blob/master/docs/cmd/scrape.md
